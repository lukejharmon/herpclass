About Long Haul

Long Haul is a minimal, long form Jekyll. It can be used as is or customized to your hearts desire.

Long Haul was created in honor of all the hard working touring bicycles that have traversed the globe time and time again. Take it for a spin it's built to last and fun to use.

## Features - SASS + COMPASS - SUSY Grid - SVG Social Icons from Customizr - Responsive Nav Menu - XML Feed for RSS Readers - Contact Form via Brace Forms - 5 Post Loop with excerpt on Home Page - Previous / Next Post Navigation - Drop Cap on posts - A More Modern Scale for Web Typography
